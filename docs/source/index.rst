MolBuster |version| documentation
====================================

MolBuster is a command line tool and Python package for checking the chemical and
physical sensibility of docked or generated molecules.

.. note::

   This project is under active development.


..note::

   MolBuster completely relies on the wonderful cheminformatics toolkit RDKit_.
   Consider citing RDKit alongside MolBuster if you use MolBuster in your research.
